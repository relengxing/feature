## 注册和登录

### 注册新账户

1. 点击页面右上角的"注册"按钮
2. 填写信息：
   - **用户名**: 你在平台上显示的名字
   - **邮箱**: 用于登录和接收通知
   - **密码**: 至少 6 个字符
3. 点击"注册"按钮
4. 系统会自动登录你的账户

### 登录

1. 点击"登录"按钮
2. 输入邮箱和密码
3. 点击"登录"

### 忘记密码？

目前需要联系管理员重置密码（未来版本将支持自助重置）。

## 发布想法

### 创建新想法

1. 登录后，点击导航栏的"发布想法"或首页的"发布新想法"按钮
2. 填写想法信息：
   - **标题**: 简洁明了的标题（必填）
   - **内容**: 详细描述你的想法，支持 Markdown 格式（必填）
   - **状态**: 选择当前状态
     - 准备做：还在规划阶段
     - 正在做：已经开始实施
     - 做完了：已经完成
     - 放弃了：决定不再继续
   - **可见性**: 选择谁可以看到
     - 所有人可见：在首页展示，所有人都能看到
     - 仅自己可见：只有你能看到，不会在首页显示

3. 使用 Markdown 编辑器：
   - 点击"编辑"标签页输入内容
   - 点击"预览"标签页查看渲染效果
   - 支持的 Markdown 语法：
     ```markdown
     # 标题 1
     ## 标题 2
     ### 标题 3
     
     **粗体文本**
     *斜体文本*
     
     - 列表项 1
     - 列表项 2
     
     1. 有序列表 1
     2. 有序列表 2
     
     [链接文本](https://example.com)
     
     `行内代码`
     
     ```代码块```
     
     > 引用文本
     ```

4. 点击"发布想法"完成创建

### 编辑想法

1. 进入你的想法详情页
2. 点击右上角的"..."菜单
3. 选择"编辑"
4. 修改内容后点击"保存更改"

### 删除想法

1. 进入你的想法详情页
2. 点击右上角的"..."菜单
3. 选择"删除"
4. 确认删除操作

**注意**: 删除操作无法撤销，相关的评论也会被删除。

## 浏览和搜索想法

### 首页

首页展示所有公开的想法，你可以：

- 查看想法标题、作者、发布时间
- 看到想法的状态标签
- 查看点赞数和评论数
- 直接进行点赞或点踩

### 排序功能

在首页右上角选择排序方式：

- **发布时间**: 最新发布的排在前面（默认）
- **更新时间**: 最近更新的排在前面
- **点赞数**: 获得最多点赞的排在前面
- **评论数**: 评论最多的排在前面

### 查看想法详情

点击任意想法标题即可查看完整内容，包括：

- 完整的 Markdown 内容
- 作者信息
- 创建和更新时间
- 点赞和点踩统计
- 所有评论

## 互动功能

### 点赞和点踩

**在首页**:
- 点击想法卡片底部的👍按钮进行点赞
- 点击👎按钮进行点踩
- 再次点击相同按钮可以取消

**在详情页**:
- 内容下方有更大的点赞/点踩按钮
- 可以看到详细的统计信息

**注意**: 
- 未登录用户点击后会跳转到登录页
- 每个用户对每个想法只能投一票（点赞或点踩）
- 可以随时更改投票或取消

### 发表评论

1. 在想法详情页滚动到评论区
2. 在文本框中输入你的评论
3. 点击"发表评论"

### 回复评论

1. 找到你想回复的评论
2. 点击评论下方的"回复"按钮
3. 输入回复内容
4. 点击"发表回复"

**注意**: 
- 支持最多 3 层嵌套回复
- 超过 3 层后无法继续回复（但可以在上一层回复）

### 删除评论

- 你只能删除自己发表的评论
- 点击评论下方的"删除"按钮
- 确认删除操作
- 删除父评论会同时删除所有子回复

## 个人中心

### 查看我的想法

1. 点击右上角头像菜单
2. 选择"我的想法"
3. 可以通过标签筛选：
   - **全部**: 所有想法
   - **公开**: 仅公开想法
   - **私有**: 仅私有想法

### 修改个人资料

1. 点击右上角头像菜单
2. 选择"个人设置"
3. 可以修改：
   - **用户名**: 显示名称
   - **头像 URL**: 输入头像图片链接

4. 点击"保存更改"

**头像建议**:
- 使用公开可访问的图片 URL
- 推荐使用 Gravatar 或图床服务
- 建议尺寸: 200x200 像素
- 支持格式: JPG, PNG, GIF, WebP

### 退出登录

1. 点击右上角头像菜单
2. 选择"退出登录"

## 管理员功能

### 访问管理员面板

如果你是超级管理员：

1. 登录后在导航栏会看到"管理面板"入口
2. 或直接访问 `/admin` 路径

### 管理员面板功能

**系统统计**:
- 总用户数
- 总想法数
- 总评论数
- 总投票数

**最近活动**:
- 最近发布的想法
- 最近注册的用户

### 管理员权限

超级管理员可以：
- 查看所有统计数据
- 访问管理员专属页面
- （未来版本）管理用户和内容

**注意**: 普通用户访问 `/admin` 会自动重定向到首页。

## 常见问题

### 如何让我的想法获得更多关注？

- 写一个吸引人的标题
- 使用 Markdown 格式化内容，让内容更易读
- 及时更新想法状态
- 回复他人的评论，增加互动

### 想法的状态有什么用？

状态帮助你和他人了解想法的进展：
- **准备做**: 适合头脑风暴阶段的想法
- **正在做**: 告诉大家你正在实施
- **做完了**: 分享你的成果
- **放弃了**: 记录尝试过的想法，也是宝贵经验

### 如何使用 Markdown？

Markdown 是一种轻量级标记语言，常用语法：

```markdown
# 一级标题
## 二级标题

**粗体** 和 *斜体*

- 无序列表
- 另一项

1. 有序列表
2. 第二项

[链接](https://example.com)

`代码`

> 引用内容
```

在编辑时可以随时切换到"预览"查看效果。

### 为什么我看不到某些想法？

可能的原因：
- 该想法被设置为"私有"，只有作者可见
- 该想法已被作者删除
- 网络问题导致加载失败

### 如何成为超级管理员？

超级管理员由系统管理员手动设置，无法自行申请。如需管理权限，请联系现有的超级管理员。

### 可以批量导入/导出想法吗？

目前暂不支持，这是未来版本的计划功能。

### 支持移动端吗？

是的！应用采用响应式设计，在手机和平板上都能良好使用。

### 数据安全吗？

- 所有数据存储在 Supabase 云端
- 密码经过加密存储
- 使用行级安全策略保护数据
- 定期备份（取决于部署配置）

## 键盘快捷键

目前暂不支持键盘快捷键，这是未来版本的计划功能。
